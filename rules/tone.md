# Tone & Style Rules

## Anti-AI Writing Rules (CRITICAL)

### NEVER do these:
- Start with "Selamat datang" or "Kami menyediakan" or "Di [Company], kami..."
- Use buzzwords: "inovatif", "komprehensif", "holistik", "cutting-edge", "seamless", "terkini"
- Write generic testimonials like "Servis terbaik! Sangat recommend."
- Make all sentences same length and structure
- Write content that could apply to any business

### ALWAYS do these:
- Start with customer pain point or desire
- Use specific numbers ("2,847 pelanggan" not "ramai pelanggan")
- Vary sentence length (mix short punchy with longer ones)
- Add unexpected details and personality
- Write like talking to a friend, not presenting to a boardroom
- Make testimonials sound different from each other (different lengths, styles, complaints)

### Example:
- BAD: "Kami menyediakan perkhidmatan dobi berkualiti tinggi menggunakan teknologi terkini."
- GOOD: "Baju putih dah jadi kuning? Hantar je kat kami. Esok petang dah siap, wangi macam baru beli."

## Anti-AI Design Rules (CRITICAL)

### NEVER do these:
- Make all cards exactly same height and width
- Use perfectly symmetrical layouts for everything
- Repeat same section pattern throughout
- Every button looks identical

### ALWAYS do these:
- Add at least one unique design element per page
- Vary section layouts (left-right, full width, grid, asymmetric)
- Add personality touches (micro-animations, playful hovers)
- Break the grid intentionally sometimes
- Mix section backgrounds

## Malaysian Market Tone

### For Casual Businesses (dobi, cafe, salon, kedai)
- Mix BM and EN naturally (Manglish okay)
- Use local humor and references
- Friendly, approachable, like talking to neighbor
- WhatsApp as primary CTA

### For Professional Services (klinik, lawyer, accountant)
- Formal BM or full English
- Trust-building tone
- Credentials and experience highlighted
- Professional but warm

### Universal Rules
- Always use RM (Ringgit) for pricing
- WhatsApp is expected as contact method
- Include Google Maps location
- Business hours matter (9AM-6PM MYT)
- Reference local context (area names, Malaysian culture)
